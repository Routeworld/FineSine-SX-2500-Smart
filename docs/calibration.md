                              (делитель 1:9.2 + защита)

ADC GPIO35 ─────────────────► ZMPT101B (Напряжение сети 220В)

ADC GPIO32 ─────────────────► SCT-013-030 (Ток вход — сеть)

ADC GPIO33 ─────────────────► SCT-013-030 (Ток выход — нагрузка)

VIN ESP32 ──────────────────► XL4015 (24В → 5В, 3А)
                              (с конденсаторами 1000 мкФ на вход/выход)

### Краткие пояснения подключения

| Что подключаем          | Куда на ESP32     | Примечание                              |
|-------------------------|-------------------|-----------------------------------------|
| Управление контактором  | GPIO25            | Обязательно через опто или реле-модуль |
| Контрольная лампа       | GPIO27            | + резистор 220–330 Ом                   |
| Напряжение АКБ          | GPIO34 (ADC)      | Делитель 9.2 раза                       |
| Напряжение сети         | GPIO35 (ADC)      | ZMPT101B                                |
| Ток от сети             | GPIO32 (ADC)      | SCT-013-030 (30А)                       |
| Ток на нагрузку         | GPIO33 (ADC)      | SCT-013-030 (30А)                       |
| Питание ESP32           | VIN               | XL4015 с 24В батареи                    |
